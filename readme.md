# Python3 Bot

A simple Python bot that answers queries based on predefined responses.

## Sample Queries and Responses

### Query: 議員?

**Response:**  
議員指的是立法會的成員，他們是被選舉或委任到立法會擔任職務的人士。根據上述提供的資訊，這裡列出了一些立法會的議員名單。

### Prompt: Who have 大紫荊勳賢?

**Response:**
